# Simple System Management CLI Tool Over TCP

Small CLI `tool` for remote system management over `TCP`. Built with `TypeScript`. Contains custom keyboard input handling. Alows users to `manage systems remotly`. Provides `integrated CLI` with custom commands that alows choosing between connected host, seeing informations about them, and disconect from server. Uses `net` built-in Node.js module for TCP server/client

# Building:

```bash
npm run build
```

`common` directory contains files only need by server
